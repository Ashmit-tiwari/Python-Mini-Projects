# 💰 Expense Calculator

## 📖 Overview

The **Expense Calculator** is a beginner-friendly Python mini project that helps users calculate their total monthly expenses and remaining savings. The program takes a user's monthly budget and multiple expense entries, then displays a summary showing the total amount spent and the amount saved or overspent.

---

## 🎯 Features

* Enter your monthly budget.
* Add multiple expenses.
* Calculate the total expense.
* Calculate remaining savings.
* Detect overspending.
* Display a clean expense summary.

## 🚀 How to Run

1. Open the project in VS Code.
2. Run `expense_calculator.py`.
3. Enter your monthly budget.
4. Enter the number of expenses.
5. Input each expense.
6. View your total expenses and remaining savings.
