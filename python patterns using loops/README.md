# 🔁 Python Patterns Using Loops

## 📖 Overview

This project is a collection of beginner-friendly Python pattern programs created to strengthen my understanding of loops and nested loops. The goal is to practice logical thinking by printing different types of patterns using `for` loops, `while` loops, and conditional statements.

---

## 🎯 Objectives

* Practice Python loop concepts.
* Improve problem-solving skills.
* Learn how nested loops work.
* Understand pattern-building logic.
* Build a strong foundation for coding interviews.

---

## 📚 Topics Covered

* `for` loops
* `while` loops
* Nested loops
* Conditional statements (`if-else`)
* Pattern printing
* Number patterns
* Star (`*`) patterns

---

## 📂 Patterns Included

* Square Pattern
* Rectangle Pattern
* Right Triangle
* Inverted Triangle
* Pyramid
* Inverted Pyramid
* Diamond Pattern
* Number Triangle
* Floyd's Triangle
* Character Patterns

> More patterns will be added as I continue learning.
